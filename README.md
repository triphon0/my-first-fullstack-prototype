# my-first-fullstack-prototype
My first full-stack project: a dynamic React + Express application showcasing API calls, user input handling, and live backend responses.
### Frontend
cd frontend
npm install
npm start

## API Endpoint
GET /secret?user=<username>

Returns a personalized success message when the correct username is provided.

## Tech Stack
- React (Frontend UI)
- Axios (HTTP requests)
- Express.js (Backend server)
- CORS (Cross-origin communication)

## Learning Goals
This project was built to understand full-stack flow from first principles:
- How frontend triggers backend logic
- How APIs transmit user input
- How state updates dynamically from server responses
